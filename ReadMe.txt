How to Run this project:

 1. Install the XAMPP Control Panel (https://www.apachefriends.org/download.html)
 2. Copy the Project folder in the XAMPP folder and paste it in the "htdocs" folder inside of the XAMPP folder.  

 3. After install, run the XAMPP Control Panel
 4. Click Start at the Apache
 5. Click Start at the MySQL
 6. Open any web browser
 7. Copy "http://localhost/Project" and paste it in the search bar 
 
8. Back to the XAMPP Control Panel
9. Click Admin button at the MySql section
10. Add the database name "cateringdata"
11. Click the database catering data, find the import section at the top of navigation bar.
12. Import file cateringdata.sql
13. Okay now the database already imported.

14. You can test and use our website happily.
